# Corsinha Store - Digital Products E-commerce

## Project Overview
A modern e-commerce website for Corsinha Store specializing in digital products. Built with a full-stack JavaScript architecture using React, Express, and JSON-based product management for easy GitHub Pages hosting.

## Architecture
- **Frontend**: React with Vite, shadcn/ui components, Tailwind CSS
- **Backend**: Express.js with in-memory storage
- **Data**: JSON-based product management for static hosting compatibility
- **Routing**: Wouter for client-side routing
- **State Management**: TanStack Query for data fetching
- **Styling**: Tailwind CSS with dark mode support

## Project Structure
